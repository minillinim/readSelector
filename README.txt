readSelector!

Overview
=========

readSelector allows you to randomly subsample a collection of reads. Reads may be paired / unpaired. Handles input / output of gzipped files

Installation
=========

Should be as simple as

    pip install readSelector


Using readSelector
========

This is avery simple script.

use:

    readSelector -h
